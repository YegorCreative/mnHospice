# Assets

Place image files in this directory.

Recommended images:
- `hero-bg.jpg`   — full-bleed hero background (landscape, people / nature)
- `give-bg.jpg`   — background for the donation section
- `team/`         — headshots for the team page (future)
- `icons/`        — any custom SVG icons

Images are referenced from the HTML/CSS using `../assets/images/<filename>`.
The `<div class="about-img-placeholder">` in index.html can be replaced with:
  `<img src="../assets/images/about-photo.jpg" alt="Our care team" />`
once a photo is available.
